
## Build

To build the application

	../scripts/build.bat
Note : Build will take aroung 10-15 mins since it will build with test cases
## Run Application

To run the application

    ../scripts/run.bat
Note: Starting application will take 5 to 10 mins
## Tests

To run the complete test cases

    ../scripts/runtest.bat
Note: Test run will take 5 to 10 mins
## Log Files
runConsoleOutput.txt - will have the console logs of build.bat

testRunConsoleOutput.txt - will have the console logs of runtest.bat

## Application logs
will be stored in  ../logs/imdb.log
